== Description ==

This plugin helps to hide the specified words in your content.

== Installation ==

1. Upload the `WordPress Hidden Words` plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin.
3. Put the word/words you want to hide between `<hide>` and `</hide>`.
4. The hidden word/words will be replaced with `**** Hidden Words ****` to readers who don't have the edit-permission.
